# Expense Tracker

A simple Python project to manage daily expenses.

## Features
- Add Expense
- View All Expenses
- Delete Expense
- Easy menu-based interface

## Technologies Used
- Python

## How to Run

1. Clone the repository:
```bash
git clone <repository-link>
```

2. Go to the project folder:
```bash
cd expense-tracker-python
```

3. Run the program:
```bash
python expense_tracker.py
```

## Example

```
Welcome to Expense Tracker!

===== MENU =====
1. Add Expense
2. View All Expenses
3. Delete Expense
4. Exit
```
